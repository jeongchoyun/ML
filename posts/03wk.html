data = pd.read_csv("healthcare-dataset-stroke-data.csv")

df = data.copy()

1. 인코딩

A. 클래스 레이블(타깃 변수) 인코딩

1) LabelEncoder()

from sklearn.preprocessing import LabelEncoder
y = LabelEncoder().fit_transform(df['stroke'].values)
y
array([1, 1, 1, ..., 0, 0, 0])
Important
  • 참고
  • LabelEncoder 클래스
    • 타깃 레이블을 인코딩하기 위한 클래스. 입력 데이터로 1차원 배열을 기대함.
    • 여러개의 특성(열)을 작업하기에는 반복작업을 해야하므로 범주형 데이터를 정수로 인코딩하는 OrdinalEncode와 판다스 데이터프레임의 열마다 다른 변환을 적용하도록 도와주는 ColumnTransformer를 이용
    • OrdinalEncode와 ColumnTransformer는 특성 변환에서 설명

2) numpy 이용

1) OneHotEncoder-ColumnTransformer

from sklearn.preprocessing import OneHotEncoder
from sklearn.compose import ColumnTransformer
X = df.drop(["stroke"], axis=1)
onehot = OneHotEncoder(drop = 'first', handle_unknown='ignore', sparse_output=False)

ct = ColumnTransformer([('onehot', onehot, X.select_dtypes(exclude = "number").columns)],
                        remainder='passthrough', n_jobs=-1)
ct
