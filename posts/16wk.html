  • hidden_layer_sizes

      

    • 은닉층 뉴런 개수.
      • 

    • array-like of shape(n_layers - 2,)

  • activation

      

    • 은닉층 활성 함수
      • 

    • 기본값 relu {identity, logistic, tanh, relu}

        

      • solver`
        • 

      • 

    • 경사 하강법의 종류
      • 

    • 기본값adam {lbfgs, sgd, adam}

        

      • lbfgs : an optimizer in the family of quasi-Newton methods. (small dataset)
        • 

      • sgd : stochastic gradient descent.
        • 

      • adam : stochastic gradient-based optimizer proposed by Kingma, Diederik, and Jimmy Ba (large datasets)
        • 

      • comparison between Adam optimizer and SGD
        • 

      • https://scikit-learn.org/stable/auto_examples/neural_networks/plot_mlp_training_curves.html#sphx-glr-auto-examples-neural-networks-plot-mlp-training-curves-py
        • 

      •

  • alpha

      

    • L2 규제의 크기
      • 

    • 기본값 0.0001
      • 

    • 값이 클수록 과적합 방지
      • 

    • https://scikit-learn.org/stable/auto_examples/neural_networks/plot_mlp_alpha.html#sphx-glr-auto-examples-neural-networks-plot-mlp-alpha-py
      • 

    • 

  • batch_size

      

    • 확률적 최적화 미니배치 사이즈
      • 

    • 기본값 auto = min(200, n_samples)
      • 

    • 

  • learning_rate

      

    • 가중치 업데이트 학습률 스케줄.
      • 

    • solver가 sgd일 때만 사용 가능
      • 

    • 기본값 constant {constant, invscaling, adaptive}

        

      • constant : ’learning_rate_init’으로 주어진 일정한 학습률.
        • 

      • invscaling : ’power_t’의 역 스케일링 지수 사용. 각 시간 단계 t에서 점진적으로 학습률 감소. effective_learning_rate = learning_rate_init / pow(t, power_t)
        • 

      • adaptive : 학습 손실이 계속 감소하는 한 학습률을 learning_rate_init로 일정하게 유지.
        
두 연속 에포크 동안 학습 손실이 tol만큼 감소하지 않거나, early_stopping이 켜져 있을 때 검증 점수가 tol만큼 증가하지 않으면, 현재 학습률이 5로 나누어집니다.
        • 

      • 

    • 

  • learning_rate_init

      

    • 초기 학습률. 가중치 업데이트 시 스텝 크기를 조절.
      • 

    • solver가 sgd 또는 adam일 때만 사용.
      • 

    • 기본값 0.001
      • 

    • 

  • power_

      

    • 역 스케일링 학습률의 지수.
      • 

    • learning_rate가 invscaling으로 설정되었을 때 유효 학습률 업데이트.
      • 

    • solver가 sgd일 때만 사용.
      • 

    • 기본값 0.5
      • 

    • 

  • max_iter

      

    • 최대 반복 횟수.
      • 

    • solver가 이 횟수만큼 반복하거나 tol에 의해 수렴이 결정될 때까지 반복.
      • 

    • 확률적 slover(sgd, adam)의 경우, 경사 하강 단계 수가 아닌 에포크 수(각 데이터 포인트가 사용되는 횟수, 전체 데이터셋을 한 번 학습하는 과정) 결정
      • 

    • 기본값 200
      • 

    • 

  • tol

      

    • 최적화를 위한 허용 오차.
      • 

    • 손실이나 점수가 n_iter_no_change 연속 반복 동안 tol만큼 개선되지 않으면, learning_rate가 adaptive로 설정되지 않는 한 수렴에 도달한 것으로 간주하고 학습 중지.
      • 

    • 기본값 0.0001
      • 

    • 

  • n_iter_no_change

      

    • tol 개선을 충족하지 못하는 최대 에포크 수.
      • 

    • solver가 sgd 또는 adam일 때만 사용.
      • 

    • 기본값 10
      • 

    • 

  • shuffle

      

    • 각 반복마다 샘플 셔플.
      • 

    • solver가 sgd 또는 adam일 때만 사용.
      • 

    • 

  • early_stopping

      

    • 검증 점수가 향상되지 않을 때 학습 중지
      • 

    • 만약 True로 설정된 경우, 학습 데이터의 10%를 자동으로 검증 데이터로 분리하고, 검증 점수가 n_iter_no_chang 연속 에포크 동안 tol만큼 향상되지 않으면 학습을 중지.
      • 

    • 분할은 계층화(stratified) (multilabel 설정에서는 제외됨).
      • 

    • False로 설정된 경우, 학습 손실이 tol이 n_iter_no_change 동안 더 이상 개선되지 않을 때 학습 중지.
      • 

    • solver가 sgd 또는 adam일 때만 사용.
      • 

    • 

  • validation_fraction

      

    • early_stopping = True 설정 시, 검증 세트 비율.
      • 

    • 기본값 0.1 (0, 1)
